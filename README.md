#### 使用前准备
1. IDE安装 **lombok** 插件！！！
2. 下载nacos，并且导入数据
#### 使用指南
1. 导入sql文件
2. 导入项目
3. 修改启动参数，指定nacos.url等参数（具体看nacos中的配置，可自行修改）
4. 启动即可
5. 导入前端[（地址）](https://github.com/yuuki80code/cooky-cloud-web)
6. 安装依赖启动即可
